В данном проекте производтся тестирование сайта Web_Tours с предварительно зарегистрированными юзерами. Тестирование будет производиться разными профилями нагрузки: Ramp-Up-Load - для выявления максимума VUs/RPS; Steady Load - для проверки стабильности системы; Peak Load - для проверки того, как система себя поведет при резком повышении пользователей; Так же для вывода метрик используется Prometheus + Grafana, для системных метрик - Node exporter; При подготовке к тестированию были оформлены документы МНТ, Отчет о проведении НТ - https://docs.google.com/document/d/1GteKlUMTgaomqaUcanB2QoKghNJ6HLhECzW5wGdGt_M/edit?usp=sharing
